Popping Heads

Taken from https://codepen.io/oliviale/pen/XGLOem

A CSS experiment for heads popping up in your slides

As done by Chen Hui Jing in this article here.

What's Going On Here?
Using CSS Animations to change the values of translateX, translateY, and rotate and then setting the animation to run infinitely.

Limitations:
There can only be a limited variety of head popping styles as compared to randomizing the location of the head each time with JavaScript like she did (read the article!).
